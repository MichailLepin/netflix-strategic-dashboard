**Function 1: create_engagement_line(watch_df: pd.DataFrame) -> go.Figure**
- Group watch_df by watch_date (resample weekly using pd.Grouper on watch_date) and subscription_plan
- IMPORTANT: watch_df does not have subscription_plan — it must be joined with users_df. Add users_df as a second parameter: create_engagement_line(watch_df, users_df). Merge on user_id to get subscription_plan, then group.
- Y-axis: mean watch_duration_minutes / 60 (hours)
- Use px.line() with color=subscription_plan
- Title: "Weekly Watch Hours by Subscription Plan"
- Handle empty DataFrame: return empty figure with annotation "No data for selected filters"

**Function 2: create_genre_bar(watch_df: pd.DataFrame, movies_df: pd.DataFrame) -> go.Figure**
- Merge watch_df with movies_df on movie_id to get genre_primary
- Group by genre_primary, sum watch_duration_minutes / 60 (hours)
- Take top 10, sort descending (highest at top in horizontal bar)
- Use px.bar() with orientation="h", color=genre_primary
- Netflix-themed color_discrete_sequence: ["#E50914", "#B20710", "#831010", "#5C0A0A", "#FF6B6B", "#FF9999", "#CC3333", "#993333", "#660000", "#330000"]
- Title: "Top 10 Genres by Watch Time (Hours)"
- Handle empty DataFrame gracefully

**Function 3: create_device_donut(watch_df: pd.DataFrame) -> go.Figure**
- Count watch sessions by device_type (value_counts)
- Use px.pie() with hole=0.4, names=device_type, values=count
- Color sequence: Netflix palette reds/grays: ["#E50914", "#B20710", "#564D4D", "#831010", "#999999"]
- Title: "Watch Sessions by Device"
- Handle empty DataFrame gracefully

**Function 4: create_rating_scatter(watch_df: pd.DataFrame, movies_df: pd.DataFrame) -> go.Figure**
- Merge watch_df with movies_df on movie_id to get imdb_rating and genre_primary
- Group by movie_id: compute mean watch_duration_minutes and count of sessions, plus first imdb_rating and genre_primary
- Use px.scatter() with x=imdb_rating, y=avg_watch_duration_minutes, size=session_count, color=genre_primary
- Title: "Watch Duration vs IMDb Rating"
- Set size_max=20 to keep dots readable
- Handle empty DataFrame gracefully

For each function, apply:
- fig.update_layout(template="plotly_dark", height=300, margin=dict(t=30, b=20, l=20, r=20))
- fig.update_layout(paper_bgcolor="rgba(0,0,0,0)", plot_bgcolor="rgba(0,0,0,0)") for transparent backgrounds that blend with Streamlit dark theme
- Remove unnecessary gridlines: fig.update_xaxes(showgrid=False) and fig.update_yaxes(showgrid=False) for cleaner look (except keep y-grid on line chart for readability)
- Compact legend: fig.update_layout(legend=dict(orientation="h", yanchor="bottom", y=-0.3)) or hide if chart is self-explanatory (bar chart categories are self-labeled)
